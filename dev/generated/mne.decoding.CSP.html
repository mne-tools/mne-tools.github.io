mne.decoding.CSP¶

class mne.decoding.CSP(n_components=4, reg=None, log=True)¶

M/EEG signal decomposition using the Common Spatial Patterns (CSP)

This object can be used as a supervised decomposition to estimate spatial filters for feature extraction in a 2 class decoding problem. See [1].

Parameters:

n_components : int, default 4

The number of components to decompose M/EEG signals. This number should be set by cross-validation.

reg : float, str, None

if not None, allow regularization for covariance estimation if float, shrinkage covariance is used (0 <= shrinkage <= 1). if str, optimal shrinkage using Ledoit-Wolf Shrinkage (‘lws’) or Oracle Approximating Shrinkage (‘oas’).

log : bool

If true, apply log to standardize the features. If false, features are just z-scored.
Attributes:

filters_ : ndarray

If fit, the CSP components used to decompose the data, else None.

patterns_ : ndarray

If fit, the CSP patterns used to restore M/EEG signals, else None.

mean_ : ndarray

If fit, the mean squared power for each component.

std_ : ndarray

If fit, the std squared power for each component.

References

[1] Zoltan J. Koles. The quantitative extraction and topographic mapping of the abnormal components in the clinical EEG. Electroencephalography and Clinical Neurophysiology, 79(6):440–447, December 1991.
