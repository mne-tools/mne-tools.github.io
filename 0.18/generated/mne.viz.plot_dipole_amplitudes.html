    mne.viz.plot_dipole_amplitudes¶

    mne.viz.plot_dipole_amplitudes(dipoles, colors=None, show=True)[source]¶

    Plot the amplitude traces of a set of dipoles.

    Parameters
    dipoleslist of instance of Dipole

    The dipoles whose amplitudes should be shown.

    colors: list of color | None

    Color to plot with each dipole. If None default colors are used.

    showbool

    Show figure if True.

    Returns
    figmatplotlib.figure.Figure

    The figure object containing the plot.

    Notes

    New in version 0.9.0.
