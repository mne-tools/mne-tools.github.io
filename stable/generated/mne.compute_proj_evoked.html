    mne.compute_proj_evoked¶

    mne.compute_proj_evoked(evoked, n_grad=2, n_mag=2, n_eeg=2, desc_prefix=None, verbose=None)[source]¶

    Compute SSP (spatial space projection) vectors on Evoked.

    Parameters:
    evoked : instance of Evoked

    The Evoked obtained by averaging the artifact

    n_grad : int

    Number of vectors for gradiometers

    n_mag : int

    Number of vectors for magnetometers

    n_eeg : int

    Number of vectors for EEG channels

    desc_prefix : str | None

    The description prefix to use. If None, one will be created based on tmin and tmax.

    New in version 0.17.

    verbose : bool, str, int, or None

    If not None, override default verbose level (see mne.verbose() and Logging documentation for more).
    Returns:
    projs : list

    List of projection vectors

    See also

    compute_proj_raw, compute_proj_epochs
