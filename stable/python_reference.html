API Reference¶

MNE for MEG and EEG data analysis

This is the classes and functions reference of mne-python. Functions are grouped thematically by analysis stage. In addition, all File I/O functions are collected in a separate section. Functions and classes that are not below a module heading are found in the mne namespace.
