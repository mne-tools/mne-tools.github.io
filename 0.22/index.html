Open-source Python package for exploring, visualizing, and analyzing human neurophysiological data: MEG, EEG, sEEG, ECoG, NIRS, and more.

Source estimation

Distributed, sparse, mixed-norm, beamformers, dipole fitting, and more. Try it

Statistics

Parametric and non-parametric, permutation tests and clustering. Try it

Machine learning

Advanced decoding models including time generalization. Try it

Encoding models

Receptive field estimation with optional smoothness priors. Try it

Connectivity

All-to-all spectral and effective connectivity measures. Try it

Data visualization

Explore your data from multiple perspectives. Try it
